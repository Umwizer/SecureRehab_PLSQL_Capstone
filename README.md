## Project Overview

SecureRehab is a PL/SQL-based database system designed to track offender rehabilitation, calculate risk scores, generate alerts, and support early-release decisions to reduce prison overcrowding in Rwanda.

## Problem Statement

Rwandan prisons face severe overcrowding and lack real-time offender progress monitoring, leading to inefficient parole decisions and high recidivism rates.

## Objectives

1. Develop a secure Oracle database for offender management.
2. Implement automated risk scoring and alerting.
3. Support data-driven early-release decisions.
4. Provide BI dashboards for authorities.
