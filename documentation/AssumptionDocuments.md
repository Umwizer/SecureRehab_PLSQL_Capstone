# SecureRehab Assumptions

1. Maximum offender age is 100 years; minimum is 18 years.
2. Gender is recorded as 'M' (Male) or 'F' (Female).
3. Offense severity ranges from 1 (low) to 5 (high).
4. Rehabilitation program duration is measured in full weeks.
5. An offender can participate in multiple programs simultaneously.
6. Alerts are generated automatically upon new offense registration or program completion.
7. Risk scores are calculated on a 0â€“100 scale and updated after offenses or program participation.
8. Early-release eligibility is determined by program completion and low-risk score.
9. All dates are stored in DD-MON-YYYY format.
10. Null values allowed only for optional fields like address or alert description.
